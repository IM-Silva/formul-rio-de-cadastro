# 📚 Projeto de Integração

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte da disciplina de **Projeto de Integração**, com o objetivo de aplicar conhecimentos em desenvolvimento de software, programação web e banco de dados.

A aplicação integra um sistema desenvolvido em Python com um banco de dados Oracle, utilizando SQL para gerenciamento e manipulação das informações.

---

## 🚀 Tecnologias Utilizadas

- Python 3
- HTML5
- CSS3
- SQL
- Oracle Database Free
- Podman
- Git
- GitHub
- Visual Studio Code

---

## ⚙️ Funcionalidades

- Cadastro de informações.
- Consulta de registros.
- Alteração de dados cadastrados.
- Exclusão de registros.
- Integração entre aplicação e banco de dados Oracle.
- Interface web para interação com o usuário.

---

## 🗄️ Banco de Dados

O projeto utiliza **Oracle Database Free** como sistema gerenciador de banco de dados.

As operações de criação, consulta, atualização e exclusão de dados são realizadas utilizando comandos SQL.

---

## 📚 Objetivos do Projeto

- Aplicar conceitos de programação em Python.
- Desenvolver uma aplicação web.
- Utilizar SQL para manipulação de dados.
- Integrar aplicação e banco de dados Oracle.
- Praticar versionamento de código com Git.
- Publicar projetos utilizando GitHub.
